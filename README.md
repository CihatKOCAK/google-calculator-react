## Run it on your computer

Clone the project

```bash
  git clone https://github.com/CihatKOCAK/google-calculator-react.git
```

```bash
  cd google-calculator-react
```

Install required packages and run

```bash
  yarn; yarn start
```
or
```bash
  npm i; npm start
```
